### To install:
```
bundle install
bundle exec rake db:create
bundle exec rake db:migrate
bundle exec rake db:seed
rails s -p 3001
```

### To consume API:

```
git clone https://github.com/kkchu791/eightball.git
webpack-dev-server
```
